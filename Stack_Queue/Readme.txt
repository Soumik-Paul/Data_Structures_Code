#include <stack>
Main functions:

push(element) → Adds element to the top.

pop() → Removes the top element (no return value).

top() → Returns reference to the top element.

empty() → Returns true if stack is empty.

size() → Returns number of elements.

#include <queue>
Main functions:

push(element) → Adds element to the back.

pop() → Removes element from the front.

front() → Returns reference to the first element.

back() → Returns reference to the last element.

empty() → Returns true if queue is empty.

size() → Returns number of elements.
